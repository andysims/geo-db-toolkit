# geo-db-toolkit

This repository is a collection of small, personal tools I’ve put together to help with day‑to‑day data, database, and GIS tasks. Most of the scripts are focused around GIS (Admin mostly) workflows, PostgreSQL/PostGIS, SQL Server, and general SQL, PowerShell, or Python utilities.

These tools are intentionally simple (for my sake).

Although this repo is primarily for my own use, others are welcome to explore it, reuse anything that helps, or adapt the scripts to fit their own workflows.

## What you’ll find here

- PowerShell utilities for working with databases, data (and data prep), and GIS Admin tasks (will likely add some Azure stuff at some point)
- SQL helpers for PostgreSQL/PostGIS and SQL Server
- Python scripts
- Miscellaneous tools for GIS and databases

## Notes

- Everything here is meant to be (mostly) easy to read, modify, and customize.
- Tools are built around and for GIS or data tasks

## Usage

Each tool is documented inside its own folder

Feel free to explore, copy, adjust, or improve anything in the repository.

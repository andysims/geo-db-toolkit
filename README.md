# geo-db-toolkit

This repository is a collection of small, personal tools I’ve put together to help with day‑to‑day data and database tasks. Most of the scripts are focused around GIS workflows, PostgreSQL/PostGIS, SQL Server, and general SQL, PowerShell, or Python utilities.

These tools are intentionally simple.

Although this repo is primarily for my own use, others are welcome to explore it, reuse anything that helps, or adapt the scripts to fit their own workflows.

## What you’ll find here

- PowerShell utilities for working with CSVs, databases, and data preparation (will likely add some Azure stuff at some point)
- SQL helpers for PostgreSQL/PostGIS and SQL Server
- Lightweight Python scripts
- Miscellaneous tools for GIS and database work

## Notes

- Everything here is meant to be (mostly) easy to read, modify, and customize.
- Most tools are built around real GIS or data tasks

## Usage

Each tool is documented inside its own folder

Feel free to explore, copy, adjust, or improve anything in the repository.
